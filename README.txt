Install Drupal instantly - IDI.
This script create virtual host and install Drupal automatically.

Installation and usage
1. You must install Drush
2. Copy the file idi.drush.inc into a .drush directory in your home directory.
For example /home/user/.drush/idi.drush.inc
3. Clear the cache Drush using the drush cache-clear drush
4. Start drush idi by the command sudo drush idi.
